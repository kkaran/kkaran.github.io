Responsive Design

Questia offers an entirely adaptive interface, that doesn't remove features when browsed on mobile devices. I would start the process with a responsive layout in mind, informed by user research and realized using wireframing tools.

Refining the Interface

Singular interfaces that provide textual information can be intrusive and cluttering. When implementing the wireframes, I would organize my syntax to factor fluid layouts.

Beyond Questia

Using a library of API's, Questia has extended it capabilities beyond its domain. I have been often tasked with creating the marketing material that support our external applications, from landing pages through app store assets.
